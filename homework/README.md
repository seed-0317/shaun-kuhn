# Homework

Copy this `homework` directory to your personal repo.  Write code such that all tests pass.

### 1. Install node and npm

```
brew install node
```

```
node -v
npm -v
```

### 2. `cd` to this directory and run install

```
npm install
```

### 3. Install Jasmine globally

```
npm install -g jasmine
```

### 4. Run all tests

```
jasmine
```
